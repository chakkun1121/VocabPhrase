# キーボードショートカットについて

## 編集ページ

| キー     | 機能 |
| -------- | ---- |
| Ctrl + S | 保存 |

## フラッシュカード

### フラッシュカードで暗記中

| キー       | 機能                                     |
| ---------- | ---------------------------------------- |
| space      | 解答を表示(キーボード解答モードオフ時)   |
| ctrl+enter | 解答する(キーボード解答時)               |
| c          | チェックを付ける、はずす(解答表示時のみ) |
| r,s        | 読み上げ                                 |
| →          | 次へ                                     |
| ←          | 前へ                                     |
